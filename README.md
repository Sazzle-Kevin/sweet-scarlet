## 🎯 Über das Projekt
Dieses Projekt wurde in einem Zeitraum von 5 Tagen von einer studentischen/Arbeits-Gruppe entwickelt. Ziel war es, eine performante und visuell ansprechende Webseite zu einem Thema (Code: Sweet Scarlet) umzusetzen.


## 🛠️ Technologien
- **HTML5** – Semantische Struktur
- **Tailwind CSS** – Utility-First CSS Framework
  

## ✨ Features
- 📱 **Fully Responsive:** Optimiert für Mobile, Tablet und Desktop.
- ⚡ **Performance:** Schnelle Ladezeiten durch optimierte Tailwind-Klassen.
- 🎨 **Modernes Design:** Clean, intuitiv und farblich abgestimmt.
